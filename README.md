Graphite
====

### Requirements
GCC 4.7 or later.  If you're using 4.4, you're a mouth breather - Dillion 


### installing
Graphite uses cmake. To build Graphite run:

```
git clone https://github.com/dillonl/graphite.git
cd graphite/
mkdir build
cd build
camke ..

```

### running

To run Graphite's adjudication:

./graphite
